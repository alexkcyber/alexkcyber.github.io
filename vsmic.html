Virtual Strategic Missile Integration Complex

There is a need to be able to perform cyber assessments on the Air Force's Strategic Missile Integration Complex (SMIC). This is difficult to accomplish due to the fact that the SMIC was created before the invention of internet.

A script was developed that automates the validation/verification of simulated weapon system circuits. The script automatically performs comprehensive tests and returns the results in a matter of seconds. This was an important effort because manual testing took many hours of tedious work and the script reduced this time greatly.
